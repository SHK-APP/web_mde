Html의 태그

<strong > = 글의 굵기를 굵게한다.
< u > = 글에 밑줄을 추가한다.
< br > = 줄을 바꾼다.
< p > = 지정된 곳을 글의 단락으로 지정한다.
< head > = 지정된 곳을 글의 헤더로 지정한다.
< img > = 이미지를 삽입한다. src = 를 붙여 출력할 이미지의 경로를 설정할 수 있다,
속성 설정으로 width=를 통해 크기를 조절하고 src=를 통해 이미지의 경로를 설정할 수 있다.
< li > = 목록을 생성한다. < ul > = li의 부모 태그, 주로 li로 설정한 리스트들의 성격 별로 정리하기 위함이다. < ol > = ordered list 숫자 라벨이 붙은 리스트로 만듬 < title > = 페이지의 제목을 설정한다. < meta charset > = 코딩 형식을 설정한다. < html > = html 문서임을 알려주는 최상위 태그 < head > = 본문을 설명하는 부분임을 알려줌 < body > = 본문임을 알려줌 < a > = 클릭하면 이동가능한 링크를 삽입한다. 속성 target = _blank 로 새 창에서 열게 하기 가능
